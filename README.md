# PokeDex
Pokedex app Up to gen9
Pokédex — Battle Type Helper
A fast, mobile-friendly Pokédex web app focused on battle strategy. Look up any Pokémon across all generations (Gen 1–9, including Scarlet & Violet) to instantly see type matchups, weaknesses, abilities, and base stats — all in one clean interface.
Live app → mexbatlab.github.io/PokeDex

Features
Pokémon Search

Search by name or Pokédex number (e.g. "Miraidon" or "1000")
Supports all 1000+ Pokémon including Paldea natives and DLC additions
Displays official artwork, types, and Pokédex number in a hero card
Tap the sprite to reveal all abilities with a description panel — hidden abilities are highlighted in purple
Type matchup breakdown (the main event):

Double weak — 4×
Weak — 2×
Strong against — 2× (offensive)
Resists — 0.5×
Double resist — 0.25×
Immune — 0×


Collapsible base stats with color-coded bars and total BST — tucked away so matchups stay front and center

Browse by Type

Visual grid of all 18 Pokémon types with custom icons
Tap any type to instantly see its full offensive and defensive matchup profile
Same ordered matchup breakdown as the Pokémon tab

Design

Dark and light mode with a toggle — remembers your preference
Installs as a PWA (Progressive Web App) — add to your home screen on iPhone or Android for a native app feel
Responsive layout that works on phones, tablets, and desktop
Smooth animations and type-colored hero sections
